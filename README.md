# Solfège Trainer

A tiny, dependency-free web app for **sight-singing** and **ear-training**
practice. Pick scale degrees, set a key and vocal range, generate a random
melody, sing it, then play it back to check yourself.

**Live:** <https://ears.niknok.space>
